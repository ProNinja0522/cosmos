Basic Concepts in Probability

In probability theory, we study how likely events are to occur.

A random variable is a function that assigns numerical values to outcomes of a random experiment.

Key Properties

  • A probability measure PPP assigns values in [0,1][0, 1][0,1].
  • The sample space Ω\OmegaΩ contains all possible outcomes.

Python: Estimate Probability via Simulation

import random

trials = 10000
successes = sum([random.random() < 0.3 for _ in range(trials)])
estimated_prob = successes / trials
print("Estimated Probability:", estimated_prob)
